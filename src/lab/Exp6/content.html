Theory

Matrix:



A matrix is defined as a system of elements arranged in a rectangular array in the form of rows and columns.The matrix is designated by a capital letter. The general form of any matrix is

A=[aij] mxn


where,
m = no. of rows, n = no. of columns and aij is the corresponding element




Matrix Basic Operations:



1. Addition and Subtraction

Two or more matrices are added with a necessary condition that the matrices are of same order. Consider two matrices A and B of the same order mxn their sum A+B is also a mxn matrix obtained by adding the corresponding elements. i.e.

A+B=(ai,j)1≤i≤m;1≤j≤n +(bi,j)1≤i≤m;1≤i≤n =(ai,j + bi,j) 1≤i≤m;1≤i≤n
2. Multiplication

The multiplication of two or more matrices is possible if and only if the number of columns of the first matrix is equal to the number of rows of the second matrix. If A is an mxn matrix and B is an nxp matrix, their product AB is the mxp matrix whose (i, j)th entry is computed by multiplying each element of the ith row of A to the corresponding elements of the jth column of B and add the result.

(AB)i,j=ai,1.b1,j+ ai,2.b2,j+….+ai,n.bn,j


3. Transpose

Consider a matrix A of order mxn, then a matrix obtained by interchanging the rows and columns of the original matrix A is known as the transpose of the matrix A and is designated as AT or A' which is of the order of nxm. i.e.

A=[ai,j] then AT=[aj,i]


4. Inverse of a Matrix

Consider a square matrix A, then the inverse of the matrix A which is designated as A-1 and is defined as

A-1=adj (A)/|A|


where A-1 = inverse of the matrix A
adj (A) = (-1)i+j X determinant of cofactor of aij
|A|= determinant of the matrix A
The A-1 matrix exists if and only if when |A|≠0

Inverse of a Matrix via Decomposition Methods

Matrix decomposition refers to the transformation of a given matrix into a given canonical form. When the given matrix is transformed to a right-hand-side product of canonical matrices the process of such decomposition is also called “matrix factorization”. Matrix decomposition is a fundamental operation in linear algebra and applied statistics used for engineering significance. The purpose of matrix decomposition involves two aspects: computational convenience and analytic simplicity. It is not feasible for most of the matrix computations to be calculated in an optimal way, such as matrix inversion, matrix determinant, solving linear system and least square fitting, thus to convert a complex matrix computations into easier tasks such as solving triangular or diagonal system will greatly facilitate the calculations.
Consider a linear system of simultaneous equations AX=B,
where A=coefficient matrix, X=variable matrix and B=constant matrix
Now there are various decomposition methods by which coefficient matrix A can be expressed are as follows:
Singular Value Decomposition
LU Decomposition
Cholesky Decomposition
QR Decomposition
Schur Decomposition and Eigen-value Decomposition

Steps to perform experiment:



1)  Start simulator by clicking on the Simulation tab.
2)  Click on the link to enter the simulator.
3)  Login to your virtual Lab account.

5)  Then select an Experiment to perform from list.
6)  After filling the necessary fields click on Submit button.
7)  Here you will get answer to verify answer from simulator.
8)  Now click on Generate Verilog Code button to generate the code.
9)  Load the generated program by clicking on Load Program tab.
10)  Enter the file name without extension.
11)  Click on Save On Server button to save program on your workspace at server.
12)  Now click on Compile tab to compile your code on server.
13)  After successful compilation, click on ok to generate the test bench.
14)  Click on checkbox to select as output registers and select its type as output from list.
15)  Enter data width (MSB=7 and LSB=0 for 8-bit output register).
16)  Click on Submit button.
17)  Click on Submit Test Bench button to submit your test bench.
18)  Click on Close This Window button.
19)  Click on Execute tab which is under Compile tab.
20)  Click on Timing Analysis tab to view Timing Analysis of input program.
21)  For Resource Utilization Report click on Resource Utilizaion Report tab.
22)  Your programs are saved on your workspace at server, which you can find under Your Programs tab.
23)  If you want to write your own code Go to File-> New and repeate the procedure from step 10 to 21.
