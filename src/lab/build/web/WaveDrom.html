Step 1. The clock

Clock is the special signal. It changes twice per cycle and can have positive or negative polarity. The following WaveJSON represent two clock signals. 
 
{ "signal" : [
  { "name": "pclk", "wave": "n......" },
  { "name": "nclk", "wave": "n....." },
  { "name": "Pclk", "wave": "P....." },
  { "name": "Nclk", "wave": "N....." }
]}
