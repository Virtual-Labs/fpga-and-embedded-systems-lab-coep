Welcome to FPGA & Embedded Systems Lab!
Introduction

In this lab we will demonstrate varilog as hardware programming language for FPGA. This lab demonstrates different arithmetic operations and logical operators, linear equations and matrix algebra, Designing of reconfigurable architecture for PID Controller etc.

List of experiments
  • Introduction to FPGA and Verilog Programming
  • Introduction to Verilog Abstraction Levels
  • Fixed Point Arithmetic Operations and Logical Operators
  • Floating Point Arithmetic Operations
  • Logarithmic Number System (LNS) Based Arithmetic Operations
  • Linear Matrix Algebra
  • Linear Equation Solver
  • Pulse-Width Modulation (PWM) Generation Using FPGA
  • Architectural Design Approach
  • Designing of Reconfigurable Architecture for PID Controller
